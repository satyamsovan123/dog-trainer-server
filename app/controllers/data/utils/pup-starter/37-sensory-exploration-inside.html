Steps

Set up this game in a different room if possible, preferably with a different floor surface. If needed, introduce a slippery surface using vinyl floor covering.

  1. Create a puppy exploration course in the room using various objects.
  2. Include items that can make noise, such as treats in a metal pan with a slightly closed lid or a stack of oven trays.
  3. Stack plastic plant pots or empty plastic bottles to add movement and a different kinetic experience.
  4. Hide treats and toys under, inside, or on top of some objects to encourage exploration.
  5. Allow Shadow to explore freely for up to 30 minutes or longer as desired.

Tips
